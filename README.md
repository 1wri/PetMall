## 一、介绍

java jsp宠物商城系统带7000字文档和20页ppt java项目ssm项目jsp项目

语言: Java 数据库：MySQL

技术栈： JSP + Spring+Spring MVC+Mybatis

系统角色：管理员、用户

管理员：登录、订单管理、客户管理、商品管理、类目管理、修改密码

用户：注册、登录、首页、商品分类、热销、新品、
加入购物车、提交订单、查看订单
